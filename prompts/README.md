# README — Prompts Padrões

Esta pasta contém prompts prontos para usar com uma IA de programação, seguindo a arquitetura:

```txt
Prompt → orquestrador.md → controle-projeto.md → protocolos → speciality → skills → execução → documentação/checkpoint
```

## Como usar

1. Abra o prompt adequado.
2. Preencha os campos entre colchetes.
3. Envie para a IA de programação.
4. Exija que ela leia `orquestrador.md`, `controle-projeto.md`, `docs/` ou `doc/`, `protocolos/`, `speciality/` e `skills/`.
5. Ao final, exija atualização de documentação e checkpoint.
