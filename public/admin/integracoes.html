Autenticação

Toda chamada da API externa usa um token Bearer no cabeçalho Authorization. O token é exibido uma única vez na emissão — guarde-o em cofre de segredos.

GET /parceiro/carteira/52998224725 HTTP/1.1 Host: seu-dominio Authorization: Bearer SEU_TOKEN Content-Type: application/json
